# FeedMark - Assignment Marking and Feedback System

Honours project for creating efficient marking workflows with reusable feedback.

## Project Overview
FeedMark is a web application designed to streamline the assignment marking process for educators. It allows markers to create marking schemes, attach reusable feedback comments to criteria, and export marks and feedback in a Canvas-compatible format.

## Features (Planned)
- Create and manage assignments with sections and criteria
- Build reusable feedback comment libraries
- Record marks with attached feedback
- Export to Canvas-compatible CSV format
- Multi-user support with moderation features
- Analytics for common feedback patterns
